
*Requirements
	-Node.js with NPM.
	-Internet Connection.

* Steps to  run the whatwhowhere app with gulp

1. clone the folder in your system
2. Open the nodejs command promt
3. Go inside the folder with command “cd folderName” 
4. Now install all the devDependecies in your system which are mentioned in package.json with the command “npm install” 
   it will take time around 10-20 minutes.
5. After the installation, run the command “npm install gulp -g”
   it will take time around 10-20 minutes.
6. Run command 'gulp' then app will be automatically loaded in browser.
